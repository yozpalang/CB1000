برای نصب اول 1 رو بزنید روی برخی موبایل ها با 2 کار میکنه . اگه با 1 نشد قبلش یک بار Uninstall کنید یعنی 3 رو بزنید.

پس از نصب از 4️⃣و 5️⃣ هم میتونید استفاده کنید.

4️⃣ لوکیشن ایران رو تغییرمیده به کشوری مثل آلمان( بهترین لوکیشن) همون قابلیت gool میشه

6️⃣ هم برای اکانت شما حجم رایگان میگیره که اکانت به warp plus تبدیل بشه.

گوشی iOS هم که فعلا هیچی.

⭐️‏اسکنر IP وارپ آپدیت شد.

یه حرکت جالبی زدم که میتونید علاوه بر گرفتن IPv4 و IPv6 کانفیگ رایگان برای Nekobox و V2rayNG و WireGuard هم بگيريد مستقیما کپی میکنید و استفاده میکنید.
قابلیت اجرا روی Termux و همچنین روی سرور رو داره و اینکه برای ویندور هم
یه اسکنر اوکی کردم که داخل گیتهاب یک فایل zip گذاشتم اونو دانلود میکنید از حالت فشرده خارج و اجراش می‌کنید.

کانفیگ ها رو کپی میکنید شاید برای paste کردن داخل V2ray و بیشتر روی Nekobox احتمالا خطا بگیرید باید داخل یک ویرایشگر وارد کنید فاصله ها رو اصلاح کنید و مجدد کپی و استفاده کنید..
توضیحات داخل صفحه ی گیتهاب روحتما بخونید. هرچی نیاز بوده رو گفتم . و نکته پایانی اینکه بهتره اول یک ای پی بگیرید بعد ، کانفیگ رو بگیرید تا خودکار ای پی روی کانفیگ ست بشه.
دیگه بیشتر از این نمیتونستم از termux کار بکشم. امیدوارم به کارتون بیاد.

وارد تنظیمات شوید.

گزینه Private DNS provider hostname را انتخاب کنید.
عبارت زیر را وارد کنید:
dot.libredns.gr
روی Save بزنید.
🔹 در اندروید پایین تر از ۹ و در صورت اتصال به Wi-Fi

بخش اتصالات یا Network & Internet را باز کنید.
وارد قسمت Wi-Fi شوید و شبکه متصل را انتخاب کنید.
گزینه‌ی Modify Network یا Advanced را انتخاب کنید.
در قسمت IP settings، حالت را به Static تغییر دهید.
در بخش DNS مقادیر زیر را وارد کنید:

DNS 2: 178.22.122.100

✅ آموزش اتصال در iOS؛ :

با مرورگر Safari به این آدرس ها بروید:

LibreDNS DoH
LibreDNS DoH: No Ads

LibreDNS DoT *
LibreDNS DoT: No Ads **

اونی که ستاره خورده اول تست کنید.

روی Allow بزنید تا پروفایل دانلود شود.
سپس وارد مسیر زیر شوید:
Settings > General > VPN & Device Management > Profile Downloaded
روی Install بزنید و مراحل را ادامه دهید تا نصب کامل شود.
پس از نصب، تنظیمات DNS شما به‌صورت خودکار روی شکن رایگان ست می‌شود.



✅ تنظیمات دستی در صورت اتصال به WiFi:

وارد Settings (تنظیمات) شوید.
وارد بخش Wi-Fi شوید.
روی آیکون ℹ️ کنار شبکه Wi-Fi متصل ضربه بزنید.
به پایین اسکرول کرده و روی Configure DNS بزنید.
گزینه Manual را انتخاب کنید.
در بخش DNS Servers، همه آدرس‌های موجود را حذف کنید و با کلیک روی Add Server آدرس‌های زیر را وارد کنید:
116.202.176.26
178.22.122.100
روی Save بزنید.
توجه: این تنظیم فقط برای همان شبکه Wi-Fi اعمال می‌شود. در صورت تغییر شبکه، باید تنظیمات را مجدد انجام دهید.

✅ آموزش اتصال در ویندوز:

وارد Control Panel شوید.
به مسیر زیر بروید:
Network and Internet > Network and Sharing Center > Change adapter settings
روی شبکه (Wi-Fi یا Ethernet) که به آن متصل هستید راست‌کلیک کرده و گزینه Properties را انتخاب کنید.
گزینه Internet Protocol Version 4 (TCP/IPv4) را انتخاب کرده و روی Properties بزنید.
گزینه Use the following DNS server addresses را فعال کنید.
مقادیر زیر را وارد کنید:
Preferred DNS server:
http://116.202.176.26
Alternate DNS server: 178.22.122.100
روی OK بزنید و تمام پنجره‌ها را ببندید.
در صورت لزوم مرورگر خود را یکبار بسته و مجدد باز کنید.
